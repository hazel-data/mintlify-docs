## Overview

Loop Returns helps brands turn returns into exchanges and retain more revenue. Connect it to Hazel to analyze your return trends — why customers return products, how often they exchange vs refund, and which products have the highest return rates.

## What Hazel pulls

Once you connect Loop Returns, Hazel has access to:

- **Returns** — every return with status, outcome, and timeline
- **Exchanges** — when customers swap for a different product instead of refunding
- **Return line items** — which specific products are being returned and why
- **Return shipments** — shipping and tracking data for returned items
- **Return outcomes** — whether each return resulted in a refund, exchange, or store credit

## How to connect

Find Loop Returns in the [Hazel marketplace](https://app.fromhazel.ai/integrations/marketplace?search=loop+returns) and click the **+** button. You'll need to paste your Loop Returns API key.

**What you'll need:** Your Loop Returns API key, found in your Loop admin settings.

## Example questions you can ask

Once your Loop Returns data is connected, try questions like these:

- "What's our return rate this month?"
- "What are the top reasons customers return products?"
- "What percentage of returns are exchanges vs refunds?"
- "Which products have the highest return rate?"
- "How much revenue are we retaining through exchanges and store credit?"
- "Show me return trends over the last 90 days"

## Good to know

- **Exchange vs refund rate** is one of the most valuable metrics here — it tells you how much revenue you're saving through Loop's exchange flow.
- Loop Returns tracks Shopify returns only — Amazon FBA returns are tracked separately through the [Amazon](/integrations/amazon) integration.
- Pair with Shopify to see return rates in the context of total orders and revenue.
- Your data refreshes automatically.
